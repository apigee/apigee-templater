# Apigee Feature Templater (aft) 🚀

**Apigee Feature Templater** is a command line tool to help manage Apigee proxy templates & proxies.

## Install the CLI
```sh
# install to use the 'aft' command
npm i apigee-templater -g
# update
npm update apigee-templater -g
```
